# Course Project - Alpha Release

## Overview
This is the Alpha Release of our project. It implements the first 4 core user stories/functional requirements.

## Prerequisites
Ensure you have the following installed on your local Windows machine:
- **WampServer (WAMP)** (Apache & MySQL)
- **PHP 8.x**
- **Composer**

## How to Build and Setup the System
1. **Clone the repository:**
   Open GitHub Desktop, go to `File > Clone repository`, and select this repository.

2. **Install Dependencies:**
